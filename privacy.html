Privacy

Privacy and security notes for this starter site.

This is not legal advice. Replace this page with a real privacy policy before collecting customer information, accepting payments, using analytics, or connecting third-party services.

Current static-site behavior

The starter site is built for GitHub Pages and does not include a backend, database, payment processing, analytics, tracking pixels, or third-party embeds.

Before collecting inquiries

  • Use a secure form service or backend with HTTPS.
  • Do not expose API keys or credentials in frontend code.
  • Collect only the information needed to answer rental requests.
  • Add a real privacy policy and rental terms.
  • Validate submissions server-side if a backend is added.

Security checklist

  • No hardcoded secrets.
  • No inline tracking scripts.
  • No fake submitting form.
  • External links should use safe link behavior.
  • Update placeholder content before launch.
